Weather Data

This data set is focused on weather data. Particularly comparing different characteristics of the weather in different cities by latitude.

Navigating

Navigate to the data visualizing through either of the below methods:

  1. Select the visualization in the dropdown box in the navigation bar. This will navigate to the respective page in your current tab.
  2. From the home screen, select "Go to visualization" to the visualization preview that you are interested in from the carousel below. This will navigate to the respective page in a seperate tab.

To view all four visualizations at once, navigate to the comparison page via the navigation bar button.
Finally, to see the data we worked with, navigate to the data page via the navigation bar button.
